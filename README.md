# Finance Dashboard

An MVP personal finance dashboard app to track recurring items and monthly cashflow.

## Tech Stack
- Frontend: Next.js + Tailwind CSS
- Backend: FastAPI
- Database: SQLite

## Getting Started

### Backend
1. \`cd backend\`
2. \`python -m venv venv\`
3. Activate virtual environment
4. \`pip install -r requirements.txt\`
5. \`uvicorn app.main:app --reload\`

### Frontend
1. \`cd frontend\`
2. \`npm install\`
3. \`npm run dev\`
